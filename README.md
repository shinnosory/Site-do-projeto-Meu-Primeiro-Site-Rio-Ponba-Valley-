# Site-do-projeto-Meu-Primeiro-Site-Rio-Ponba-Valley-
Site criado para o projeto Meu Primeiro Site, apresentando informações sobre a ODS 5 – Igualdade de Gênero. Desenvolvido com HTML e CSS, o site explica metas, desafios, causas da desigualdade e formas de contribuir, além de indicar materiais para aprofundamento. Uso exclusivamente educativo.
